The Train Writes Context

Each cell is an independent LLM context window. The train carries a rolling context — as it visits cells, it:

1. Reads the cell's POML data (pollen)
2. Writes tokens into the cell's context
3. Refracts through the stained-glass filter
4. Ripples outward — new context windows propagating

Click any cell to view its context window.
Press ▶ RUN TRAIN to start the locomotive.
