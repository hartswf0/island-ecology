0.0 TERMINAL DIRECTIVE Replace typing with control surfaces
Old operation: humans type into passive database fields.
New operation: agents execute typing/retrieval/scheduling rules.
Clinician role: read output + press button (sign/approve/export).
1.0 SCRIBE LEVER (Audio → Note) Control: in-room device
Place device in exam room.
Speak normally (no keyboard).
Review structured draft on desktop.
Click Sign.
Output: SOAP/structured note draft. Gate: clinician sign-off.
2.0 DIGESTION LEVER (Data → Insight) Control: purple indicator
Open schedule panel; look for purple indicator.
Click to view evidence-backed suggestions.
Click Add to chart (after review).
In inbox: batch-sign green-coded routine items.
Output: care gaps/codes/refill drafts. Gate: clinician review.
3.0 CONDUCTOR LEVER (Speech → Orders) Control: mic button
Press mic.
Say: “Order ___.”
Confirm staged card matches intent.
Click Export to EHR queue.
Rule: if ambiguous, ask one question—do not guess meds.
4.0 FRONT DOOR LEVER (SMS → Scheduling) Control: text thread
Patient texts day (“Tuesday”).
System offers 3 real slots.
Patient picks (“10 AM”).
System locks slot + confirms.
Hard stop: emergency keywords → halt + “Dial 911.”

Warnings & Contraindications
How this system fails (and how you prevent it)
BLACK BOX RISKHIGH (if unmanaged)
Warningsdo not skip
  • Transcript-as-law: if the UI treats drafts as truth, clinicians will comply to avoid audits.
  • Automation overreach: never send patient messages or finalize orders without explicit approval.
  • Ambiguity gambling: medication name confusion must trigger a question, not a guess.
  • Alert fatigue: staging too much creates blindness. Stage only what matters.
Contraindicationsdo not deploy like this
  • No evidence trails (no “why” for each staged action).
  • No human gates (auto-finalizing notes/orders/messages).
  • No emergency stop words (front door keeps scheduling during red flags).

About this Label

This is a nutrition-label style interface for an architectural shift in healthcare software: System of Record → System of Intelligence. The label makes the product readable in under a minute.

What this is scope
  • A compact spec for deploying an autonomic agent layer that runs routine clinic functions.
  • Four levers: scribe, digestion, conductor, front desk.
  • A strict stance: evidence trails + human gates are mandatory.
