# Island Ecology
## The Archipelago of Epistemic Tooling

This repository houses a suite of generative, analytical, and structural interfaces developed for mapping and tearing apart conceptual entities. Below is a map to navigate the distinct territories comprising this project.

### Navigation Map

#### 1. [Stained Contexts](./STAINED%20CONTEXT/index.html)
The anatomical register. These interfaces use Voronoi partitioning and Transclusion algorithms to create severe, structural "Glass Fields." They do not flex; they hold and synthesize generative inputs within constrained spaces. See the [Stained Context README](./STAINED%20CONTEXT/README.md) for more info.
- Core: `stained-glass.html`, `stained-atlas.html`, `stained-skill.html`

#### 2. [Bees on a Train](./BEES%20ON%20A%20TRAIN/world-analyzer.html) 
The core data layer. This directory is responsible for parsing raw `.md` narratives into structured, timeline-based JSON manifests. It provides tools to map entities, skills, and events across different conceptual "worlds." See the [Bees on a Train README](./BEES%20ON%20A%20TRAIN/README.md) for data schemas.
- Core: `world-analyzer.html`, `skill-analyzer.html`

#### 3. [MCP-Ripples](./MCP-RIPPLES/README.md)
The media generation suite. This folder contains automated FFmpeg bash scripts and python pipelines designed to build serialized video grids and cinematic layouts from base assets, outputting linear and tiled (`.mp4`) video matrices.

#### 4. The Root Directory Prototypes
Loose tools acting as testing grounds for the Ripple AI context extraction and Epistemic Crushers.
- `index.html`: The Prompt Console & Experimental Setup
- `index_of_indexes.html`: The Global Nested Node Viewer
- `achilles-rlm-engine.html` / `cheap-rcc.html`: Generative interfaces mapping out recursive loop mechanisms.
