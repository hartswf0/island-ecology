Mobile Operator Manual

This interface is a spatial control mechanism. It does not ask you to write a story first. It asks you to instantiate a field, inject a stone, append a site, and materialize the consequence.

Generate = Instantiate Integrate = Inject Conjure = Append Make = Materialize
clean model

What this system is actually doing

You give the machine a world seed. It builds a sparse site graph. You inject a local event into one site. That event changes site state, neighbor pressure, and language output. A selected line can then be compiled into a 360 photosphere image.

Prompt architecture

1. Genesis prompt concept → topology

The concept becomes a sparse graph of concrete sites. In demo mode this comes from local theme banks. In live mode it asks the model to produce a connected ecology.

2. Stone prompt event → state delta

Your text is classified as SHIFT, GOAL, or OBSTACLE, then routed into one node as an event payload.

3. Node mind prompt site → utterance

The selected site speaks locally: fragment, worldtext, stage, intent, target neighbor, and gallery lines.

4. Ecologist prompt field → direction

A field-level conductor biases the graph toward harmony, counterpoint, or stabilization.

5. Photosphere prompt line → image

The selected gallery line becomes a rigid scene brief for one coherent 360 environment with grounded materials and no collage junk.
