# STAINED CONTEXT

This directory contains the **"Stained Glass"** interfaces. They are structural mechanisms built upon HTML Canvas and Graph/Voronoi rendering. They do not represent a biological body; they represent absolute constraints on how text and generative concepts are partitioned and displayed.

## The Mechanisms

### The Scaffolds (Atlas, Clavicle, Sternum, Rib)
These files act as foundational frameworks. They isolate inputs into distinct Voronoi cells. They enforce juxtaposition without allowing the synthesis of the partitioned concepts. 
- **Atlas**: Tracks lineage via Xanadu methods.
- **Clavicle**: Suspends and relates entity data.
- **Sternum**: Assembles core generative prompts.
- **Rib**: Constrains the boundaries of context playback.

### The Actuators (Lat, Bicep, Radius)
These files execute lateral movement and force action. They drag foreign context into the local frame alongside the main sequence text.
- **Lat**: Facilitates lateral context synchronization between left and right panes.
- **Bicep**: Direct injection and actuation controller.
- **Radius**: Maps entities radially upon the canvas.

### The Assemblies (Carpal, Bone, Tarsal, Skill, Plus)
These are granular, high-intensity operational hubs that introduce advanced rendering states.
- **Carpal**: Formatter for document mode rendering.
- **Bone**: Assembles Hyperlegible states and the Quiver tool array.
- **Tarsal**: Manages lateral accounting via wide, horizontal scroll interfaces.
- **Skill**: Tracks and displays the "Ghost Quiver" logic.
- **Plus**: Intersects database files with visual world manifests.

## Usage
Launch `index.html` to view the **Anatomical Register**, which serves as the localized map and launching point for these interfaces.
