ACHILLES // QUICKSTART

This system works best when you read it as a field instrument. Build the topology, drop one stone, choose the right line, then make or scout the photosphere.

1. World

Generate the field before expecting speech

Use GENERATE for real mode or DEMO for local rehearsal. Real mode is allowed to remain silent until a topology and first stone exist.

2. Stone

Treat the prompt as a disturbance, not a chat

A stone is one causal event injected into one site. The graph then translates that into worldtext, gtext, and image variants.

3. Images

Images belong to lines, not to the site in general

Each node keeps a small gallery. Pick the line you want, then use MAKE IMAGE. REROLL remakes the current line only.

4. 360

Scout a place from inside it

Use ENTER 360 once a line has an image. In immersive mode you can move between sites, change lines, and capture a scout frame.

Look

Style conditioning is now one-button simple

The LOOK button cycles image prompt profiles: scout, archival, operatic, and nocturne. This changes the image brief without changing the world state.

Mobile

Use the drawer buttons when space is tight

On smaller screens, INSPECT and LOG open the side panels as drawers. The header keeps only the critical controls visible.
