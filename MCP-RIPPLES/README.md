# MCP-RIPPLES
## Media Orchestration and Output

This directory is an output and processing toolkit for generating, compiling, and finalizing video assets and multimedia generated throughout the *Island Ecology* project lifecycle.

### Tools & Contents
- `make_videos.py` / `make_videos_v2.py` / `make_videos_v3.py`: Generative pipelines and FFmpeg automation scripts for tiling, sequencing, and stitching together 3x1 and 5x1 grid videos.
- `run_ffmpeg*.sh`: Native shell scripts for performing fast video transforms, scaling, and audio sync tests locally.
- Generative Source Output: Stored artifacts like `output_linear.mp4`, `output_3grid_seq.mp4`, etc., representing the finalized visualizations.
